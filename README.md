# Antigravity Core

Framework profesional para estandarizar proyectos usando Antigravity.

## Objetivos

- Mantener consistencia entre proyectos.
- Reducir improvisación.
- Mejorar calidad técnica.
- Estandarizar workflows.
- Reutilizar capacidades.

## Estructura

### Rules
Restricciones absolutas globales.

### Walkthroughs
Procesos operativos y metodologías.

### Skills
Capacidades especializadas reutilizables.

## Filosofía

- Simplicidad.
- Escalabilidad.
- Mantenibilidad.
- Reutilización.
- Arquitectura racional.
